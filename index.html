OilApp

Multi-Platform Application for Optimizing Olive Oil Production Processes

Final Year Project | 4inTechnologie | February 2025 - August 2025

About the Project

OilApp is a comprehensive multi-platform application dedicated to digitalizing Tunisian olive oil mills. This innovative solution transforms traditional olive oil production processes through advanced technology and artificial intelligence.

Key Features

Electronic Weighing System: Automated weight tracking and management
Digital Invoicing: Streamlined billing and payment processing
Customer Tracking: Comprehensive client relationship management
AI-Powered Energy Optimization: Predict costs and resource consumption
Machine Learning Algorithms: Random Forest & Gradient Boosting with R² = 0.95

Technical Achievements

  • 95% Accuracy: Energy consumption prediction model
  • Real-time Processing: Instant data analysis and reporting
  • Secure Architecture: Enterprise-grade security implementation
  • Cross-Platform: Works seamlessly on mobile and web

Technologies Used

Flutter Django PostgreSQL Python Machine Learning Random Forest Gradient Boosting AI Optimization RESTful API
Project Impact

OilApp revolutionizes the olive oil industry in Tunisia by providing mill owners with intelligent tools for cost optimization, process automation, and data-driven decision making. The AI-powered energy prediction system helps reduce operational costs while improving production efficiency.
